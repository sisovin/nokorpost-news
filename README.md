## âœ¨ Getting Started

1. **Clone the Repository**
    ```bash
    git clone https://github.com/sisovin/nokorpost-news.git
    cd nokorpost-news
    ```

2. **Install Dependencies**
    ```bash
    npm install
    # or
    yarn install
    ```

3. **Run the Development Server**
    ```bash
    npm run dev
    # or
    yarn dev
    ```

4. **Open in Browser**
    - Visit [http://localhost:3000](http://localhost:3000)
